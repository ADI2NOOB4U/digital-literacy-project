Project Overview

This project helped me understand what digital literacy really means in today’s world. It’s not just about using technology, but also about using it in a smart, safe, and responsible way. Through this project, I learned how to manage my online identity, communicate properly on digital platforms, and work with others using online tools. I also became more aware of cyber threats and how to protect myself from them. Overall, this project gave me practical knowledge that I can apply in my daily digital life.

Tasks Completed
Task 1: Infographic on Digital Literacy

I created an infographic to explain the main ideas of digital literacy in a simple and visual format. This helped me break down complex concepts into easy-to-understand points.

Task 2: Digital Portfolio Setup

I built a basic digital portfolio where I can showcase my skills, projects, and achievements. This will help me create a professional online presence in the future.

Task 3: Coding & Collaboration Platforms

I explored platforms like GitHub and learned how developers collaborate and manage code online. This gave me an idea of how teamwork works in the tech field.

Task 4: Email Etiquette

I learned how to write proper professional emails, including using the right format, tone, and clear communication. This is important for both academic and professional environments.

Task 5: Cybercrime Awareness

I studied different types of cybercrimes such as phishing, hacking, and online scams. I also learned how to stay safe online by following basic security practices.
